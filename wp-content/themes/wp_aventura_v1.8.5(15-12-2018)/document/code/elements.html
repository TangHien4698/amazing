Aventura WordPress Theme

Learn about how to use some frequent elements

Introduction

There are a lot of elements which are default elements of Visual Composer and shortcodes developed by TemPlaza. It is required to install Aventura and Visual Composer plugin before using these elements. Then you click Add Element to open element table:

Revolution Slider

You can decide which slider displayed in a specific page

Revolution Slider

  • Before adding this element to your page, firstly, you have to create your slide. Go to Dashboard --> Slider Revolution to create new slide. You can also import our demo slides easily using zip files included in demo data folder.
  • If you use One click Installation, these demo slides are imported automatically.

Latest Tour

With this element, you can show travel tours which have been recently created. There are two Latest Tour types:

* Type 1: Select to show tours by Categories or name

* Type 2: Add Links and Description. You can also select to show tours by Categories or name

HOW TO CREATE A TOUR

Tour Kind

With this element, you can introduce your tour in easy way. In this element, it is necessary to add Background image, enter Title, select Tour Category/Name to be shown and Total Tour Items shown

Before adding this element to your page, firstly you need to have tours and tour categories available. Go to Dashboard -->Tour to create tours.

Counter

This element allows you to show real statistics in an attractive way to impress your visitors. You can easily select the icon and image. Font Awesome Icons are used. Eg: fa fa-users

Search Tour

You can add this element to create search destination, date, location, and duration for your tour

Before adding this element to your page, firstly you need to have tours available. Go to Dashboard --> Tour to create tours.

Hotline

This element allows you to show out your phone number and email address that clients can contact with service provider.

Background Type: You can select to add a background image or color

Testimonials

This element is used to add your advertisement information including Image and Text, or display tourist's reviews and ratings.

Testimonials is added in Tour section. You can select to show testimonials by Tour Category or Tour Name.

Before adding this element to your page, firstly you need to have tours available. Go to Dashboard --> Tour to create tours.

Banner Slider

This element includes 2 parts:

  • Banner Slider Settings: Add Background Image and configure sliders: Auto Play, Timeout, and Loop or not.
  • Banner Slider Item Settings Used to configure each slide. You can select to make Tours be slides or create new slides using Custom Post.
Before adding this element to your page, firstly you need to have tours available. Go to Dashboard --> Tour to create tours.

Services

Show your available services easily with this element. In this element, you can select to add an Image or Icon. Font Awesome Icons are used.

Articles

Show what you can see and enjoy when you travel. In this element, enter Title, Posts per page, order and select Post Category.

Before adding this element to your page, firstly, you need to create posts. Go to Dashboard --> Posts --> Add New to get started.
