Tour

VIEW DEMO

1. Create List & Grid Tour

Method 1:

Step 1: Go to Appearance --> Menus

Step 2: Add a Custom Link to your menu.

Step 3: Add the tour link. The tour link structure: your_url/tour. Eg: wordpress.templaza.net/aventura/tour

Step 4: To configure Booking form, you go to Dashboard --> Theme Option --> Tour Setting --> List & Grid. To add Send Enquiry form, you create it in Contact section, then, go to Theme Options --> Tour Settings --> Contact

Method 2:

Step 1: Go to Appearance --> Menus

Step 2: In Tour, select to add All Tour

Step 3:

Step 4: To configure Booking form, you go to Dashboard --> Theme Option --> Tour Setting --> List & Grid. To add Send Enquiry form, you create it in Contact section, then, go to Theme Options --> Tour Settings --> Contact

  • Refer to Tour Settings option for detail information
  • Before create this page, firstly, you need to create your tours.

2. Create Single Tour

  • Step 1: Go to Dashboard --> Tour --> New Tour
  • Step 2: Enter your title and content, tour category.
  • Step 3: Add other information in Tour Option and upload Itinerary Image You can select one of 3 types: Daily Tour, External Tour and Special Tour. Then, you can enter detail tour information such as: Date, Available days, Time Schedule, Destinations, Tour Combos, Price, Gallery, Location as well as detail itinerary.
  • Step 4: To configure single tour, you go to Dashboard --> Theme options --> Tour Settings --> Tour Detail and Contact

TIP: In case you want to show Tour Category Description, you need to add description by going to Dashboard --> Tours --> Tour Categories and add tour description for each tour category.

TOUR SETTINGS

Book & Contact Options: This option is to configure the price box, booking form and contact form.

  • Booking Head Option: Used to hide/show price box in the viewing tour. If you select Default, the configuration will be same with configuration in Theme Options > Tour Settings > Tour Detail In this option, to show/hide the price box for the viewing tour, select Custom > Move to Booking Head Display Option > Select to Show/hide
  • Booking Form Option: Used to hide/show booking form in the viewing tour. If you select Default, the configuration will be same with configuration in Theme Options > Tour Settings > Tour Detail. In this option, to show/hide the booking form for the viewing tour, select Custom > Move to Booking Form Display Option > Select to Show/hide
  • Contact Option: Used to hide/show the contact form in the viewing tour. If you select Default, the configuration will be same with configuration in Theme Options > Tour Settings > Tour Detail. In this option, to show/hide the contact box for the viewing tour, select Custom > Move to Booking Form Display Option > Select to Show/hide

Tour Info: This option is to set up content of a tour.

  • Tour Type: There are 3 types available:
    • Daily Tour: For this type, you can select a day or multiple days for your tour. You can also select specific dates.
    • External Tour: In this type, you can add an external link for your tour. From this link, the user can book and checkout instead of having these actions on your website.
    • Special Tour: This tour is normally conducted in one day.
  • Departure Time: Add time session for a tour
  • Duration Label/Duration Value: This is the time a tour is available. Duration Label is normally texts while Duration Value will be the value of Duration Label.
  • Destination/Tour Guide: Select the destination and tour guide for the tour.
  • Manage People? Total People of Tour: Select this box to enter total tourists for a tour. This total will be applied daily and time session. Eg: We have 2 tours:
    • Tour 1: From 30/5 to 31/5, Maximum people: 10. It means that there will be 10 people for date 30 and 10 people for date 31.
    • Tour 2: In 30/5, 2 time sessions (Morning and Afternoon), 10 people allowed: Mean that 10 people for Morning and 10 people for Afternoon.
  • Add Combos: Add Combos for each group of people. For each combo, it is required to add total people allowed. Eg: a combo of 2 people, in total people box, it should be 2
  • Adult/Child Price/Discount: Enter price and discount percentage
  • Gallery: Image collection which will appear on top of the tour in front end.
  • Address/Location/Introduction of the Location: The specific destination of the tour
  • Itinerary: The itinerary of the tour.

3. Create Tour Cart page

This theme has 2 available systems: Custom and WooCommerce. For Custom system, it means that you use custom Tour Cart and Tour Checkout page created by TemPlaza. For WooCommerce system, you will use the Cart and Checkout page of WooCommerce.

Below is the details on how to create Custom pages including Tour Cart, Tour Checkout and Tour Confirmation:

Step 1: Go to Page --> Add New and create a new default page named Tour Cart

Step 2: Add shortcode [tour_cart] to this page.

4. Create Tour Checkout page

Step 1: Go to Page --> Add New and create a new default page named Tour Checkout

Step 2: Add shortcode [tour_checkout] to this page.

5. Create Tour Confirmation page

Step 1: Go to Page --> Add New and create a new default page named Tour Confirmation

Step 2: Add shortcode [tour_confirm] to this page.

4. Checkout With WooCommerce

Aventura supports Checkout via WooCommerce. To enable this function, you go to Theme Option > Booking Options > Enable WooCommerce Integration

To checkout successfully via this WooCommerce method, make sure to satisfy requirements below:

  • Make sure that you have WooCommerce installed.
  • Make sure WooCommerce pages are created and set up them to correct places. To learn on how to create these pages, you go here
  • Although Cart Page of WooCommerce is used, it is still required to set up Tour Cart page Theme Options > Tour Settings > General

How To Create:

Step 1: You go to Dashboard --> Pages --> Add new to create new page.

Step 2: Select Default Template as its template.

Step 3: Click Backend Editor to edit your site with Visual Composer

Step 4: Add different elements to your page

Step 5: Add your home page to menu

Look at the below preview image to see which elements are added in this page.


Create all the modules used in this page:

You can click on the modules to see their configuration

  • Google Map
  • Contact Form
1. Map

Step 1: Go to Google Map. Search for your location.

Step 2: Click on Menu Icon and select Share or embed map

Step 3: Select Embed tab and copy the given iframe

Step 4: Go back your newly created Contact page in dashboard and paste the iframe to Map box

2. Contact Information

You can edit or change your contact information including address, email and phone in Contact page.

3. Contact Form

Make sure that the Contact Form 7 is installed.

  • Step 1: You go to Dashboard --> Contact --> Add New and create a contact form.
  • Step 2: You can use Default form or add new form given this in theme. To use theme form, you replace the default form with the new form below: 
    
                                    <p>    [text* your-name placeholder "Name"] </p>
                                    <p>
                                        [email* your-email placeholder "E-mail"] </p>
                                    <p>
                                        [text your-subject placeholder "Subject"] </p>
                                    <p>
                                        [textarea your-message placeholder "Message..."] </p>
                                    <p class="btn btn-color">[submit "Send now"]</p>
TIP: Refer to Contact Form 7 Documentation for more information.

How To Create:

It is required to have WooCommerce

After installing this plugin, you will see Woocommerce Setup Wizard . Just follow step by step to create these pages.

TIP: WOOCOMMERCE SETUP WIZARD

1. Installing pages again:

If you skipped the wizard or want to re-install the WooCommerce pages, go to WooCommerce > System Status > Tools and use the page installer tool.

TIPs:

2. Add Products

To add new products, go to Dashboard > Products > Add New

TIPs:

3. Settings

  • To set up information related to Address, products, shipping, checkout, account and email, you can go to WooCommerce > Settings
  • To setup sidebar for your shop page, go to Theme Options > Shop Options and Appearance > Widgets to add widgets to your sidebar.
