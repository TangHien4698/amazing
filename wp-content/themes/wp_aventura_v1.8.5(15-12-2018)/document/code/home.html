learn how to create demo home page!

Home Version 1

How to create:

Step 1: You go to Dashboard --> Pages --> Add new to create new page.

Step 2: Select Template Home Default as its template. Go to Templates for more information.

Step 3: Click Backend Editor to edit your site with Visual Composer

Step 4: Add different elements to your page

Step 5: Add your home page to menu

Look at the preview image to see some element added in Home 1 .

Create all the elements used in this page:

You can click on the elements to see their configuration

  • Revolution Slider
  • Search Tour
  • Services
  • Destinations
  • Counter
  • Feature Tour
  • Hotline
  • Latest Tour
  • Banner Slider
  • Articles
  • Testimonials

Info You can refer to Elements section to learn more about these elements.
