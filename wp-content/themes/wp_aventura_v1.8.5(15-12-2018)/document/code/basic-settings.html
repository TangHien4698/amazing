Aventura - Basic Settings

Learn about how to create Menu, Widgets and Footers

Tutorials

1. Aventura - Templates

In Aventura theme, there are 3 built-in templates supported:

  • Template Default: This is a default one in WordPress. When you create a default page, you can see Page General Options option:
    • Row Option: Select Container if you want to make your page boxed. Select No Container to display your page in full width.
    • Padding Top/Bottom: Add the space between your content and the top/bottom of the page.
    • Background Color: Select page color.
    • Header Option:
      • Default: Use default header type which is selected in Theme Options -> Header Options
      • Custom: Select a header type for your page.
    • Breadcrumb Option
      • Default: Use default settings in Theme Options --> Breadcrumb Options
      • Custom: Configure the breadcrumb for the editing page.
    • Newsletter Page Option
      • Default: Use default settings in Theme Options --> Newsletter
      • Custom: Configure the newsletter form for the editing page.

  • Template Home Default: Used to create Home Pages (Home 1,2,3,4).
  • Template Home Slide: Used to create Home Slide page.
  • Template Login: Used to create Login page.
  • Template Register: Used to create Register page.
  • Template Lost Password: Used to create Lost Password page.

2. Visual Composer

Visual Composer is a required plugin which allows you to build different layouts easily. You can edit your page in Backend Editor

Activate Page Builder

To start using Visual Composer, go to Page --> Add new and click on Backend Editor

Basic Usages

a. Add Elements: To add new elements, just click on Add Element

b. Row Settings and Column Settings

Row Settings - General:

  • Parallax: You can select to use parallax feature or different style here. In case, parallax feature is turned on, there is an option for you to add parallax image.
  • Row Type: There are 2 row types: Full-width and Boxed.
  • Overlay Parallax: This option allows you to add overlay color when parallax feature is used.

Row Settings - Design Option: When you edit Row Settings or Column Settings, you can edit Design Option

  • Padding: You can add padding for each element here.
  • Border: You can create borders to wrap your element by adding values here.
  • Margin: Similarly, you can set margin by adding values here.

c. Edit/Duplicate/Remove Elements:

Refer to VISUAL COMPOSER DOCUMENTATION

Menus

1. How To Create Menus


Step 1: You can manage main menu by going to Appearance--> Menus.

Step 2: Then you click Create Menu and type menu name.

Step 3: You can generate menu items from left side and drag the generated menu at the right side to sort it. After that, next task is to assign menu to theme locations.

Step 4: Click Save

TIP: Have a look here if you are still confused: WordPress Menu User Guide

Footer Settings

1. Footer Options

To configure the footer options, you go to Appearance --> Theme Options --> Footer Options. Here you can upload your footer background. This option includes 2 sub options:

  • Footer content: This option allows you to divide your footer into different columns and select width for each column. Total width for all columns should be equal or smaller than 12
  • Footer bottom: change or edit copyright.

2. Demo Footer

In demo, footer is divided into 4 equal columns. Navigate to Appearance --> Widgets to add different widgets to 4 footers. All widgets used in demo footer are listed below:

  1. Contact Info widget: In this widget, you are able to add your detail contact information including Address, Phone and Email
  2. DW Twitter widget: To use this widget, make sure that you get DW Twitter plugin installed. This widget requires you to add Consumer Key & Consumer Secret. Log in to Twitter Developer page, then create a new application to get this information.
  3. Visual Editor widget: Make sure Black Studio TinyMCE widget installed. This widget allows to add and edit your content in Text Editor normally.

Posts

Below information provides you with foundations of how to create a post with categories and tags.

A post contains 2 basic elements: Post Categories and Post Tags., so that you can manage all your posts easily. This part will introduce you to post categories and how to create them.

Categories: To add new post categories, navigate to Dashboard --> Posts --> Categories

Tags: To add new tags, go to Dashboard --> Posts --> Tags and add tag name.

To create a post, you follow steps below:

Step 1: Go to Posts --> Add new

Step 2: Add your content, then choose categories and tags.

Aventura comes with 6 different post formats: Standard, Gallery, Video, Audio, Link and Quote. After installing this plugin, you are able to choose what post format you'd like from the tab options about the text editor, and insert necessary content.

  • Standard: With this format, you are only able to add texts into your posts.
  • Gallery: Add bundle of images to your post with this format.
  • Video: You can add Video URL (YouTube or Vimeo)or Embed code here
    • How to get Embed code:
  • Audio: Similarly, you can add the Audio URL directly or add embed code here. To get these embed codes, you can go to soundcloud.com and search for your favorite audio.
  • Link: For this post type, this link will be shown in your detail post instead of your content.
  • Quote: In this type, you can add different quotes, just enter Source name, Source URL, & Content
