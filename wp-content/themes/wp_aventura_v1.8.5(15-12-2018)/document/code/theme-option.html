Aventura WordPress Theme

Learn about useful theme options

Theme Options

Aventura provides a collection of different options that make it easy to set up your site with this theme. To get these options you go to Dashboard --> Theme Options

Now you can have a quick look at these options below.

General Options

This option includes 4 options: Favicon, Loading Config, Layout, Background

  • Favicon : This option is applied for changing favicon on your site. In this option, you can upload your favicon image
  • Loading Configure : This option is applied for Loading. You can see that you can hide or show loading when you access to your site
  • Background: this option allows you to set a stunning image in order to avoid dullness during loading time.

Header Options

This option allows you to select the header type for your site. Firstly, you select header type. Secondly, go to the corresponding header option to configure.

  • Header Select Type : Select Header Type
  • Header Type 1 Options: This is the configurations for Header Type 1
  • Header Type 2 Options: This is the configurations for Header Type 2
  • Header Type 3 Options: This is the configurations for Header Type 3
  • Header Type 4 Options: This is the configurations for Header Type 4
  • Header Type 5 Options: This is the configurations for Header Type 5

Breadcrumb Option

In this option, you can change your breadcrumb options on all pages

Newsletter

Insert your newsletter form to any default page easily with this element.

Footer Options

This option includes 2 parts:

  • Footer Content: This option is applied for Footer. In this option, you can select footer type, configure number of columns, background for footer.
  • Footer Bottom: This option is footer Bottom. You can change Copyright, footer logo and link

Theme Color

Change your theme color easily in this option. You can also change color of review button, Book Now button and Sale Off flag

Typography

This option is used to change font type for any positions in your site

Blog Option

Blog general

This option is applied for Category page (not detail posts). You can select to show/hide some information such as: Sidebars, Date, Comment, etc.

Blog single

This option is applied for detail posts. You can select to show/hide some information such as: Sidebars, Date, Comment, etc.

Booking Option

These options are used to configure booking form

Payment Options

This option is applied for Payment option which allows you to turn on or off payment method via Paypal as users book tours

Tour Setting

This option includes 4 options: General, Detail, Contact and List & Grid

  • General : This option includes all options for tour part in general.

    Before configuring this option, you have to create Tour Cart page, Tour Checkoutpage, Tour Confirm page, and Wishlist page. Go to Tour for more information.

  • Contact : This option can allow you to select contact form for tour
  • Tour Detail: Includes options to configure for detail tour
  • List and Grid:

Branch Settings

This option is used to set up Branches page and Branch Detail

  • Branch List:
  • Branch Detail: This option can allow you to set up detail branch page.

Shop Setting

This part is used to set up your Shop. There are 2 options:

  • Shop List: This option is to configure your MAIN shop page. You can upload breadcrumb image, set up sidebar, number of products and product columns.
  • Shop Detail: This option is to configure Shop Detail page. In this option, you can also upload breadcrumb image, select sidebar and how you want to view the product gallery which can be Horizontal or Vertical.

Login and Register

Here are options to configure Login, Register and Lost Password

Custom CSS

This is an advanced option allowing you to add some extra configurations such as changing color, font size, etc by using CSS codes.

404 Page

This option allows you to edit your 404 page

Demo Importer

This option allows you to import demo data just in one click. Refer to Importation for more information.
