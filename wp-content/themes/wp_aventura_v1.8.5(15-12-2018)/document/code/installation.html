Basic Requirements

1. Theme Requirements:

  • PHP 5.4 or newer
  • Mysql 5.x
  • WordPress installed

2. Plugin Requirements:

After successful installation it is highly recommended to install/activate all plugins delivered with the theme. Some plugins are REQUIRED (mentioned below). Those plugins MUST be installed. Recommended plugins are optional.

Installation

Unzip the downloaded theme package. You will see 4 folders inside. You can refer to File Structure to learn more.

Via FTP
  • Unzip aventura.zip in theme folder of theme package.
  • Upload the extracted folder in previous step into /wp-content / themes/.
  • To activate the theme, go to Appearances --> Themes> and activate the installed child theme.

Via WordPress
  • Navigate to Appearances--> Themes
  • Click Add New button and then hit the Upload Theme button
  • Click on Browser to upload aventura.zip in Theme folder and click Install Now.
  • Click Activate to activate the installed themes.
  • Activate required and recommended plugins: After successful installation, it is highly recommended to install/activate all plugins delivered with the theme. Some plugins are REQUIRED. Those plugins MUST be installed. Recommended plugins are optional.
  • After installing recommended plugins, go to Plugins >> Installed plugins, then activate all installed plugins

TIP: Reference this page on how to correctly install a WordPress theme: WordPress Codex - Using Themes

- Congratulations! You have installed Aventura theme successfully. Now, there are two ways for you to To use this theme easily, you can import demo data and follow this document to set up your website.

Importation - How to import demo data

Aventura is a collection of modern and beautiful WordPress Themes. Both developers and non-trained users can use it. Aventura has already taken advantage of One-click Installation that allows you to install our demo data easily than ever. With a single click, you can get your site looked like Aventura demo.

RECOMMENDATION: You should install all plugins so that demo data can be imported completely.


GET STARTED:

Step 1: Navigate to Theme Options > Demo Importer.

Step 2: Click Import Demo button to start importation.

Step 3: Just wait few minutes before all done.
