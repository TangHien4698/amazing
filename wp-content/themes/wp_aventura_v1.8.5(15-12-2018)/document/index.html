File Structure

The Aventura is an excellent theme for any tours related business such as travelling blog, tour book. With this theme, you can write articles and posts with ease.

Unzip the file (.zip) downloaded from marketplace where you bought this theme. Extracted files will have following structure:

    ROOT/
        ├── demo-data/
        │   ├── aventura_content.xml             -  Demo content XML file.
        │   ├── aventura_theme_options.json.xml       -  Demo theme option settings.
        │   ├── slider-home.zip             -  Demo slider.
        │   ├── slider-home-1 (2,3,4).zip             -  Demo slider.
        ├── documentation/
        │   ├── code/                    
        │   ├── index.html               -   Help file
        ├── theme/
        │   ├── aventura.zip            -  Theme file - this file should be uploaded.
        │   ├── aventura-child.zip            -  Chile theme file - it is optional to upload this file.
        └── readme.txt
